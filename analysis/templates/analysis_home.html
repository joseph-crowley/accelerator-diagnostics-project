Core Functionalities

  • Run Management: Initiate, monitor, and terminate data acquisition runs with ease. View Runs | Start a New Run
  • Data Analysis: Apply filters, anomaly detection, and custom scripts to your data. View Data Records
  • Visualization: Experience data through interactive graphs, heatmaps, and more.

How to Use

Navigate to Run Management to start a new data acquisition run. Once a run is active, you can monitor it in real-time and make necessary adjustments.

Use the Data Analysis tools to deep-dive into your data. You can apply pre-defined or custom filters to get the insights you need.

The Visualization tools allow you to see your data in various formats. Interact with the data to gain deeper insights.
